# 미래학원 출결 시스템 업데이트 로그 (2026-04-09)

오늘 작업은 키패드 반응 속도 개선(렉 해결)과 SMS 발송 기능의 백그라운드 안정성 확보에 집중되었습니다.

## 1. 키패드 반응 속도 최적화 (Lag Fix)
- **증상**: 번호 입력 후 OK를 누르면 1초 이상의 지연 발생.
- **원인**: 번호 입력 시마다 매번 Firestore 네트워크 통신을 시도함.
- **해결**: 
  - `onSnapshot`을 이용한 **실시간 학생 명단 로컬 캐싱** 구현 (입력 즉시 확인).
  - **Optimistic UI** 적용: 서버 응답을 기다리지 않고 화면에 먼저 처리 결과를 표시.
  - 결과적으로 1초 이상의 지연을 **0.01초 수준**으로 단축.

## 2. SMS 백그라운드 생존 보장 (24/7 Persistence)
- **증상**: 앱을 백그라운드로 보내거나 '뒤로 가기'로 종료하면 문자가 발송되지 않음.
- **해결**:
  - **Foreground Service** 도입: 상단 알림 바에 고정되어 시스템에 의해 종료되지 않도록 조치.
  - **자동 실행**: 앱 실행 시 즉시 백그라운드 감시 서비스가 작동하도록 설계.
  - **Task Persistence**: 사용자가 최근 앱 목록에서 앱을 지워도 서비스가 유지되도록 `stopWithTask="false"` 설정.

## 3. 안드로이드 14 호환성 및 안정성 강화
- **증상**: 최신 안드로이드(14 이상) 기기에서 앱 실행 즉시 튕김 현상 발생.
- **원인**: 안드로이드 14의 강화된 포그라운드 서비스 및 알림 권한 정책 미준수.
- **해결**:
  - `app.json`에 `FOREGROUND_SERVICE`, `DATA_SYNC`, `POST_NOTIFICATIONS` 권한 명시.
  - 앱 시작 시 **알림 권한 요청(POST_NOTIFICATIONS)** 로직 추가.
  - 서비스 시작 전 안정화를 위한 지연 시간(2초) 및 에러 핸들링 보강.

## 4. 웹 브라우저 발송 보조 기능
- **내용**: 태블릿/맥북 크롬 브라우저 상에서도 발송 버튼 클릭 시 문자 앱을 여는 시도를 하도록 보안 차단 우회 로직 적용.
