# 미래학원 출결 시스템 업데이트 로그 (2026-04-10)

오늘 작업은 백그라운드 SMS 발송의 **연속성 보장**과 **기기 설정 최적화**에 집중되었습니다.

## 1. 백그라운드 서비스 안정화 (Polling Fallback)
- **증상**: 화면이 오랫동안 꺼져 있으면 Firestore 실시간 연결이 끊겨 문자가 발송되지 않음.
- **해결**: 
  - 기존의 실시간 감시(`onSnapshot`) 외에, **30초 주기 강제 확인(`getDocs`)** 로직을 추가했습니다.
  - 연결이 절전 모드에 빠지더라도 30초 이내에 미발송된 문자를 강제로 찾아내 발송합니다.
  - 이로써 네트워크 환경이 불안정한 상황에서도 발송 누락을 최소화했습니다.

## 2. 관리자 화면(Dashboard) UI 보강
- **서비스 상태 표시바**: 대시보드 상단에 실시간 감시 서비스가 작동 중인지 확인 가능한 상태 바를 추가했습니다.
- **배터리 설정 안내**: 안드로이드 14 이상의 강력한 절전 기능을 방지하기 위한 설정 가이드를 하단에 배치했습니다.

## 3. 권장 기기 설정 (중요)
코드 수정만으로는 안드로이드 OS의 강제 절전을 완벽하게 제어할 수 없습니다. 안정적인 24시간 작동을 위해 다음 설정을 확인해 주세요.

1.  **배터리 제한 없음**: 설정 > 앱 > 미래학원 > 배터리 > **'제한 없음'** 선택
2.  **절전 예외 앱 등록**: 설정 > 배터리 > 백그라운드 사용 제한 > **'절전 예외 앱'**에 추가 (제한 없음 설정 시 목록에서 안 보일 수 있으나 이미 예외 처리된 것임)
3.  **데이터 무제한**: 설정 > 앱 > 미래학원 > 모바일 데이터 > **'백그라운드 데이터 사용 허용'** 켬
4.  **화면 유지 (추천)**: 개발자 옵션 > **'충전 중 화면 켜짐 유지'** 켬 (전원 연결 시 네트워크 안정성 100% 확보)
